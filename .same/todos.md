## Реализованные функции

### 1. Мобильное меню
- Гамбургер кнопка на мобильных
- Slide-in меню слева
- Все категории и контакты

### 2. Поиск товаров
- Модальное окно с поиском
- Живой поиск по названию и бренду
- Картинки товаров в результатах

### 3. Корзина
- Slide-in панель справа
- Управление количеством (+/-)
- Удаление товаров
- Подсчет общей суммы

### 4. Анимации
- IntersectionObserver для скролла
- Плавное появление секций
- Задержки для эффекта каскада

### 5. Страницы товаров
- Детальное описание
- Характеристики
- Похожие товары
- Responsive изображения
